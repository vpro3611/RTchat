Decoupled Logic Flow

How the new components interact without touching 'Send Message' logic

The Reply Chain
1
Controller receives { content, parentMessageId }
2
UseCase fetches parent from MessageRepo
3
UseCase saves new message to MessageRepo
4
UseCase saves link to MessageReplyRepo

Key Advantage

By keeping this separate, we avoid the ERR_HTTP_HEADERS_SENT issues which were likely caused by overlapping logic or multiple responses in the modified SendMessage path.
